IpAddres

Projeto para cálculo de ações referentes a endereçamento IP.

O projeto realiza os cálculos e mostra as seguintes informações para um IP válido: 1 -Classe 2 - Máscara 3 - Endereço de Rede 4 - Endereço Inicial utilizável da Rede 5 - Endereço Final utilizável da Rede 6 - Endereço de Broadcast 7 - Verifica se o IP é válido ou não 8 - Verifica se o IP é reservado para LAN (Rede Interna)

  • Data: Setembro 2018
  • Disciplina: Redes de Computadores 2
  • Categorias: Linguagem de Programação e Redes.
